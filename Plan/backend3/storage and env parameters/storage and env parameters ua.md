# Сховище та env-параметри

> Переклад з оригіналу `None ai/working/backend3/storage and env parameters/storage and env parameters.md`. Структура і сенс збережені, граматику виправлено.

## Сховище та кеш

Сховище має містити вивід кожної сесії у форматах docx і pdf. Воно має бути доступне в GUI, щоб користувач міг завантажити файл, видалити його, побачити, з якої сесії він походить, коли був створений, та іншу корисну інформацію.

Також у сховищі програми має бути кеш усіх прикріплених файлів, які потім були стиснуті та збережені з поточним id у базі даних.

## Env-параметри

Усі env-параметри мають бути доступні для перегляду на сторінці «Settings» в GUI застосунку. Тут має бути можливість переглядати, змінювати, видаляти, додавати та редагувати наступне:

- HuggingFace API ключ
- Файл user style (має бути можливість писати його у форматі markdown з попереднім переглядом)
- Останні сесії з останніми виводами

А також, можливо, інші опції, які будуть додані пізніше.
