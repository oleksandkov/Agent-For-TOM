Хід роботи

Текст опису виконаної роботи, включає код, результати вимірювань, графіки залежності часу сортування від розміру масиву для кожного з алгоритмів.

Опис повинен містити порівняльну таблицю часу виконання та обґрунтування вибору оптимального алгоритму для кожного з тестованих наборів даних.

Висновки

Текст висновків: основні результати, аналіз ефективності, практичне застосування.
