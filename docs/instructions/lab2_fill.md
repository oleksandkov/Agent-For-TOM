# Заповнення шаблону лабораторної роботи №2 (`working/templates/lab2-template.py`)

Тобі буде передано ОДНОЧАСНО (5 елементів):
1. **Цей файл** — як заповнювати.
2. **Шаблон** — `working/templates/lab2-template.py` із плейсхолдерами.
3. **Додаткові матеріали користувача** (опційно).
4. **Файл стилю/поведінки користувача** (опційно) — якщо не порожній, дотримуйся його стилю.
5. **Параметри** (опційно, інакше дефолтні з global_instructions):
   - `length`: short (500-1000 слів) | medium (1000-1700) | long (1700-2500) | large (2500+)
   - `hardness`: school | graduate | bachelor
   - `image_mode`: none | references | full

## 1. Твоя роль

Асистент викладача. Лабораторна робота №2 — **«Дослідження лінійних структур даних: стек, черга, однозв'язний та двозв'язний список»**. Якщо користувач вказав іншу тему — бери її.

Стиль: академічний, нейтрально-діловий. Без «я»/«ми».

## 2. Структурні ВІДМІННОСТІ від Lab1 (КРИТИЧНО)

1. **Порядок секцій**: Контрольні запитання → **перед** Завданнями (не після!).
2. **Секція «Варіанти»** — тільки в Lab2. Заголовок ліворуч жирним (не по центру).
3. **Фіксовані імена** вихідних файлів: `Lab_Template_Lab2_Style.docx` / `.pdf`. Не змінюй.
4. `create_docx(filename)` / `create_pdf(filename)` — **без дефолтного значення** (не додавай `filename="..."`).
5. **Імпорт `TA_LEFT`** з `reportlab.lib.enums` — потрібен для `style_body_bold`. Не видаляй.
6. **`Spacer(1, 14)`** після «Мета роботи» у PDF — навмисний відступ, не прибирай.
7. **Логіка шляхів** у `__main__`: `os.path.dirname(os.path.abspath(__file__))` — не заміняй на захардкоджений шлях.
