# Спеціальні інструкції для Лабораторної роботи №1

**Тема**: Дослідження методів сортування в масивах (бульбашкове, вибором, вставленням, Шелла, швидке сортування; порівняння часової складності).

## Вимоги до полів JSON:

1. **`goal`** (Мета):
   - Одне речення, що починається з інфінітива («дослідити принципи роботи алгоритмів сортування масивів та експериментально порівняти їхню часову складність...»).

2. **`general_info`** (Теоретичні відомості):
   - Академічний опис методів сортування: бульбашкового, вибором, вставленням, Шелла та швидкого сортування.
   - Оцінка складності алгоритмів O(n²), O(n log² n), O(n log n).
   - Поняття найкращого, середнього та найгіршого випадків.
   - **Заборонено використовувати Markdown-форматування** (списки, заголовки, bold, italic тощо).

3. **`tasks`** (Завдання):
   - Пронумерований список з 3-5 пунктів у наказовому способі (наприклад: 1. Реалізувати алгоритми... 2. Згенерувати тестові масиви... 3. Виміряти час... 4. Побудувати графік... 5. Зробити висновки).

4. **`control_questions`** (Контрольні запитання):
   - 3-5 теоретичних запитань щодо суті та складності методів сортування (наприклад: про принцип роботи Шелла, про найгірший випадок швидкого сортування, про стабільність сортування).

5. **`bibliography`** (Література):
   - 2-3 джерела за стандартом ДСТУ 8302:2015 (наприклад, праці Кнута, Кормена, Седжвіка).
